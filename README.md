> Utils about uscc(统一社会信用代码工具方法).

## Install

```shell
$ npm install uscc-utils
# or
$ yarn add uscc-utils
# or
$ pnpm add uscc-utils
```

## Usage

```js
import { validate } from 'uscc-utils'

validate(`123456789012345678`) // true
```
